PORTFOLIO

Each project I've worked on in the past year or so has served as a means to learn or become further acquainted with a technology.

Frogger Clone

This is a copy of the classic frogger game. Move with directional keys and avoid the beatles.

Educational purpose: This was one of my projects from my Udacity coursework. The objective of this project is to learn and apply object oriented programming.

Tools used: HTML, CSS, Javascript

Google Maps Project

This app utilizes Google Maps and Wikipedia APIs to display a few museums along with their summary Wikipedia articles.

Educational purpose: To learn mvc frameworks, ajax requests, and learn more jQuery and Javascript skills.

Tools used: HTML, CSS, Javascript, jQuery, KnockoutJS, Google Maps API, Wikipedia API

Etch A Sketch

First choose the sketching mode and canvas dimensions. Then hover your mouse over the canvas to doodle.

Educational purpose: Practice event driven jQuery and Javascript. (A side benefit was getting a tad better at using CSS's display property to position elements.)

Tools used: HTML, CSS, Javascript, jQuery

Jasmine Testing

I was given a dummy site and some conditions to test. Using the Jasmine testing suite, I wrote tests that would look for errors in the provided website. The main difficulty of this project was ensuring that tests were strong and very rigorous.

Educational purpose: Get exposed to application testing for failures.

Tools used: Javascript, Jasmine testing suite

Web Optimization Project

For this project I had to optimize a site and make it render at 60fps, reduce layout refreshes, and get a Google Pagespeed score above 90.

Concepts learned: Critical Rendering Path, render blocking, asynchronous loading of external Javascript sources
